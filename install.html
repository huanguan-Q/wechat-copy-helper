📦 安装步骤

1

打开Chrome扩展管理页面

在Chrome浏览器地址栏输入以下地址：

chrome://extensions/

或者通过菜单：Chrome菜单 → 更多工具 → 扩展程序

2

启用开发者模式

在扩展管理页面右上角，找到"开发者模式"开关并启用它

开关变为蓝色表示已启用

3

加载扩展程序

点击"加载已解压的扩展程序"按钮

在弹出的文件选择器中，选择 wechat-copy-helper 文件夹

点击"选择文件夹"完成安装

4

确认安装成功

扩展应该出现在扩展列表中，名称为"微信文档归档助手"

确保扩展开关为启用状态（蓝色）

浏览器工具栏应该出现扩展图标

🔐 激活与授权

A

如何激活

点击浏览器工具栏中的扩展图标，打开弹窗，在底部找到“许可证”区域：

  • 输入购买后收到的许可证密钥（License Key），可选填购买邮箱
  • 点击“激活”，首次激活会在本设备生成一个实例（Instance）
  • 换设备使用时，请先在当前设备点击“解绑”，再在新设备“激活”
B

本地缓存说明

扩展会将以下信息保存在本地（Chrome Storage/浏览器 localStorage）：

  • licenseKey、licenseInstanceId、licenseEmail
  • licenseValid、licenseStatusText
  • licenseLastCheckedAt（上次校验时间）、licenseFailureCount（近期失败次数）

这些数据仅用于本地显示授权状态与减少频繁校验，不会上传到第三方。

C

验证失败的友好提示

  • 网络错误：请检查网络或稍后重试（若您尚未配置代理域名，预览模式下出现此提示属正常）
  • 401/403：代理未授权或 API Key 配置错误，请联系开发者
  • 422：许可证无效或实例不匹配，建议检查密钥、或先解绑后再激活
  • 429：请求过于频繁，请 1 分钟后重试
  • 5xx：服务暂不可用，请稍后重试
